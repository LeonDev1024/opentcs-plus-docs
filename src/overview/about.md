# 关于 OpenTCS Plus

## 项目简介

OpenTCS Plus 是基于 OpenTCS 核心思想构建的企业级 AGV 调度系统。在保留 OpenTCS 稳定调度内核的同时，提供了更现代化的架构、更友好的用户界面和更强大的功能扩展。

## 项目背景

OpenTCS 是一个优秀的开源 AGV 调度系统，但在实际应用中，我们发现：

- 架构相对传统，难以满足现代企业级应用的需求
- 用户界面不够友好，学习成本较高
- 功能扩展性有限，定制化开发困难

基于这些痛点，我们开发了 OpenTCS Plus，旨在：

- 提供现代化的技术架构，便于维护和扩展
- 打造友好的用户界面，降低使用门槛
- 提供丰富的功能模块，满足不同场景需求
- 保持与 OpenTCS 核心思想的兼容性

## 核心优势

### 1. 现代化架构

- 采用 Spring Boot 微服务架构，模块化设计
- 前后端分离，支持独立部署和扩展
- 基于 Vue3 的现代化前端框架

### 2. 稳定可靠

- 保留 OpenTCS 稳定的调度内核
- 经过充分测试的算法实现
- 完善的异常处理和容错机制

### 3. 易于扩展

- 模块化设计，功能解耦
- 插件化驱动框架，支持多种设备
- 丰富的 API 接口，便于二次开发

### 4. 用户友好

- 直观的可视化界面
- 完善的文档和示例
- 丰富的配置选项

## 技术特点

### 后端技术栈

- **Spring Boot** - 企业级应用框架
- **Spring Security** - 安全认证框架
- **MyBatis Plus** - 持久层框架
- **MySQL/PostgreSQL** - 关系型数据库
- **Redis** - 缓存和消息队列

### 前端技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全的 JavaScript
- **Vite** - 下一代前端构建工具
- **Pinia** - 状态管理库
- **Element Plus** - 组件库

## 适用场景

OpenTCS Plus 适用于以下场景：

- 仓储物流 AGV 调度
- 工厂内部物料运输
- 自动化产线调度
- 智能仓储管理
- 其他需要 AGV 调度的场景
